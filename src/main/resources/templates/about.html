О проекте

Курсовой проект по разработке веб-приложений на Java Spring Boot

Описание проекта

Агрегатор новостей — это веб-приложение, разработанное с использованием современных технологий Java Spring Boot, предназначенное для сбора, обработки и представления новостей из различных источников.

Проект реализует полноценную систему управления новостным контентом с поддержкой ролевой модели, интеграцией с RSS-фидами и использованием ИИ-сервисов для автоматической обработки контента.

Основные возможности

Ролевая система

Три роли пользователей: Администратор, Редактор и Читатель с различными уровнями доступа

RSS-интеграция

Автоматический импорт новостей из RSS-фидов с поддержкой множественных источников

ИИ-обработка

Автоматическая категоризация и генерация кратких сводок статей с помощью ИИ

Поиск и фильтрация

Мощная система поиска по содержимому и фильтрация по категориям

Персонализация

Сохранение избранных статей и настройка персональных предпочтений

Безопасность

Spring Security для аутентификации и авторизации с защитой от основных угроз

Используемые технологии

Backend
  • Java 17+
  • Spring Boot 3.2
  • Spring Security
  • Spring Data JPA
  • Hibernate
  • MySQL Database
Frontend
  • Thymeleaf Templates
  • Bootstrap 5
  • Bootstrap Icons
  • JavaScript (ES6+)
  • CSS3
  • Responsive Design

Архитектура

Проект построен по архитектурному паттерну MVC (Model-View-Controller) с четким разделением слоев:

  • Контроллеры — обработка HTTP-запросов и управление потоком данных
  • Сервисы — бизнес-логика приложения
  • Репозитории — доступ к данным через Spring Data JPA
  • Сущности — модели данных с JPA-аннотациями
  • Представления — Thymeleaf-шаблоны для отображения данных

Тестирование

Проект включает комплексное тестирование различных уровней:

  • Unit-тесты — тестирование отдельных компонентов
  • Integration-тесты — тестирование взаимодействия компонентов
  • Property-based тесты — проверка универсальных свойств системы
  • Security-тесты — проверка безопасности и авторизации
